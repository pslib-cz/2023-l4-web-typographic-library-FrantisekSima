Text

STL chanhes natural behavior of tags <p>, <b>, <a> and <code>

Examples

Every tag <p> has different font-size, line-height and margin then normally.

Every tag <b>, which makes text bold has different font-wight and letter-spacing then normally.

Every tag <a>, which makes text Hypertext links has it's color changed based on a state.

Every tag <code>, which takes text directly from code like this example, has different background-color and border settings.

Buttons

This is basic button format of STL. Buttons behavior is same and only thing changing is its color to fit darker and lighter theme of potencial background. If you want to use this button format all you have to do is add class="button button__basic" or class="button button__basic--dark" on your tag <a>

The reason STL has two classes to describe a button is becouse class="button" describes basic behavior of button and the secound class makes each button different. You can see button with only class="button" as a third example.
