Description

Welcome to Shaman Typographic Libraray. This project was created due to a school assignment and is supposed to simplify begining of other projects becouse by using this CSS library u will already have some of the tags and functions predefined.

Titles

This is how titles look in STL, you can use them by using basic <h1> up to <h6>. They have different font-size and letter-spacing compared to regular behavior.

Text

STL chanhes natural behavior of tags <p>, <b>, <a> and <code>

Examples

Every tag <p> has different font-size, line-height and margin then normally.

Every tag <b>, which makes text bold has different font-wight and letter-spacing then normally.

Every tag <a>, which makes text Hypertext links has it's color changed based on a state.

Every tag <code>, which takes text directly from code like this example, has different background-color and border settings.
